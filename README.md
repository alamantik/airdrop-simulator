# GTA5 Airdrop Tool

Интерактивный симулятор airdrop-зон для GTA 5 RP: карта, таймер фаз, рисование и экспорт готовых PNG для стримов и разборов.

**Открыть инструмент:** [alamantik.github.io/airdrop-simulator](https://alamantik.github.io/airdrop-simulator/)

## Возможности

- **Зоны airdrop** — готовые точки и свои зоны с перетаскиванием по карте
- **Круги R1 / R2** — настраиваемые радиусы, включение и выключение отдельно
- **Симуляция фаз** — таймлайн 0:00–15:00, фазы 2 и 3, обратный отсчёт до конца фазы
- **Инструменты на карте** — обзор, кисть, текст, дистанция от игрока, метки залазов с видео
- **Экспорт PNG** — выбор зума тайлов (Z4–Z8) и ширины картинки до 12000 px
- **Шаблоны** — несколько пресетов, автосохранение, импорт и обмен кодом шаблона

## Горячие клавиши

| Клавиша | Режим |
|--------|--------|
| `V` | Обзор |
| `B` | Рисование |
| `T` | Текст |
| `R` | Дистанция |
| `X` | Залаз |
| `Ctrl+Z` / `Ctrl+Y` | Отмена / возврат |

## Данные

Всё хранится локально в браузере (шаблоны и автосохранение). Сервер не используется — можно работать офлайн после первой загрузки карты.
